## 远程使用方法

## classify.py

- 遍历指定的输入图片目录（"./photos"），根据图片横版或竖版方向分类，并将转换后的 WebP 格式图片分别保存到横版（"./landscape"）和竖版（"./portrait"）对应的输出目录中。

```bash
cd /vol1/1000/home/random-pic-api && \
curl -sL script.meimolihan.eu.org/py/classify.py | python3 -
```

```bash
cd /vol1/1000/home/random-pic-api && \
curl -sL gitee.com/meimolihan/script/raw/master/py/classify.py | python3 -
```
